# 佛教经典阅读网站：三页面完整版

本版包含三个页面：

1. 佛教经典：`index.html`
2. 伊泽书院：`yize.html`
3. 新增佛教经典：`new-classics.html`

## 本次修改

- 去掉“佛教经典”标题下方的说明文字。
- 将“菩提书院”改名为“伊泽书院”。
- 顶部导航同步改为：佛教经典｜伊泽书院｜新增佛教经典。

## 目录数据

所有目录保存在：

`data/catalog.json`

## 上传PDF

请把所有佛教经典 PDF 文件上传到：

`pdf/`

文件名需要与 `data/catalog.json` 中的 `url` 完全一致。  
例如：

`pdf/42章经.pdf`

## 显示规则

每个页面首页只显示前10条，点击“更多”后弹窗显示该页面全部目录。

## 搜索

顶部搜索框会搜索三个页面的全部目录。
